# PlayerCountBot
 Simple Discord bot to count players in server

 ## appsettings.json example

 ``` json
 {
    "DiscordSettings": {
        "Token": "steam token api"
    },
    "MinecraftSettings": {
        "IpAddress": "ip-address-of-game-server",
        "Port": port-of-rcon-server,
        "RconPassword": "rcon-password"
    },
    "ArkSettings": {
        "IpAddress": "ip-address-of-game-server",
        "Port": port-of-rcon-server,
        "RconPassword": "rcon-password"
    },
    "ConanSettings": {
        "IpAddress": "ip-address-of-game-server",
        "Port": port-of-rcon-server,
        "RconPassword": "rcon-password"
    },
    "RustSettings": {
        "IpAddress": "ip-address-of-game-server",
        "Port": port-of-rcon-server,
        "RconPassword": "rcon-password"
    }
}
 ```
